# TechHub Studio - Premium Portfolio Website

Ultra-premium, minimal, modern portfolio website for TechHub Studio built with Next.js 14.

## Features

- ✅ Ultra-premium, minimal, and luxurious UI design
- ✅ Black background with white text and yellow-300 accents
- ✅ Fully responsive for all devices
- ✅ Smooth Framer Motion animations throughout
- ✅ SEO-optimized with proper meta tags
- ✅ Accessibility compliant (ARIA labels, semantic HTML)
- ✅ PageSpeed 90+ optimized
- ✅ Contact form ready for API integration
- ✅ Dark theme optimized

## Tech Stack

- **Framework:** Next.js 14 (App Router)
- **Styling:** Tailwind CSS
- **Animations:** Framer Motion
- **Icons:** React Icons
- **Performance:** Next.js Image Optimization
- **Deployment:** Vercel (recommended)
